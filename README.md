# molgraph
> Collection of tools to train graph neural networks.


# Requirements



1. `git clone https://github.com/chembl/ChEMBL_Structure_Pipeline.git`
    `pip install ./ChEMBL_Structure_Pipeline`
    
2. `conda install -c conda-forge rdkit` or `pip install rdkit-pypi`

3. `pip install git+https://github.com/bp-kelley/descriptastorus`

4. `pip install fastai`
